# CINEMATIC STORYTELLING WEB DESIGN SYSTEM

## Objective

Create a premium cinematic landing page.

Focus ONLY on:

- Layout
- Visual hierarchy
- Spacing
- Composition
- Animation placement
- Typography scale
- Image positioning

Do NOT focus on content generation.

---

# Design Direction

The website should feel like:

- Netflix documentary page
- Luxury travel website
- Museum exhibition website
- Cultural storytelling website

The experience should be immersive and visually driven.

---

# Layout Structure

## Section 1 — Hero

Height:
100vh

Layout:

- Fullscreen background image/video
- Dark gradient overlay
- Centered focal area
- Floating navigation at top

Composition:

- 70% visual
- 30% text

Visual priority:

1. Background media
2. Main headline
3. Supporting text
4. CTA

---

## Section 2 — Editorial Split

Layout:
Two-column asymmetrical grid

Desktop:

- 60% image
- 40% content

Mobile:

- Stacked vertically

Large spacing between sections.

---

## Section 3 — Full Width Visual Break

Layout:
Single fullscreen image

Purpose:
Create breathing space and visual impact.

Minimal text.

---

## Section 4 — Alternating Story Grid

Layout:

Row 1

- Image Left
- Content Right

Row 2

- Content Left
- Image Right

Row 3

- Image Left
- Content Right

This creates storytelling rhythm.

---

## Section 5 — Impact Numbers

Layout:

Horizontal metrics row

Structure:
Number
Label

Large typography.

No cards.

No shadows.

Minimal separators.

---

## Section 6 — Cinematic Gallery

Layout:

Masonry Grid

Features:

- Different image heights
- Uneven composition
- Edge-to-edge visuals

Images should dominate.

---

## Section 7 — Feature Spotlight

Layout:

Fullscreen section

Split:

- Large visual focus
- Supporting content area

Visual should occupy 70% of space.

---

## Section 8 — Timeline

Layout:

Vertical center timeline

Elements appear on scroll.

Generous whitespace.

---

## Section 9 — Quote Section

Layout:

Centered content

Large quotation.

Minimal surrounding elements.

Strong typography focus.

---

## Section 10 — Final CTA

Layout:

Fullscreen section

Centered content.

Minimal design.

Large spacing.

---

# Spacing System

Section Padding:
120px–200px

Container Width:
1280px max

Content Width:
720px max

Whitespace:
Extremely generous.

Never make sections feel crowded.

---

# Border Radius

Use:
16px–32px

Avoid:
Sharp corners.

---

# Shadows

Minimal.

Prefer depth through:

- Contrast
- Blur
- Layering

Not through heavy shadows.

---

# Typography Hierarchy

Hero:
72px–96px

Section Titles:
48px–64px

Subtitles:
24px–32px

Body:
16px–18px

Large vertical spacing between text groups.

---

# Motion Design

Use:

- Fade In
- Parallax
- Scroll Reveal
- Image Zoom
- Text Reveal
- Smooth Scrolling

Avoid:

- Bounce
- Elastic
- Excessive rotation
- Flashy effects

Motion should feel cinematic and premium.

---

# Visual Principles

Prioritize:

1. Large imagery
2. Strong hierarchy
3. Editorial composition
4. Asymmetrical balance
5. Generous whitespace
6. Immersive scrolling

The website must feel like a visual story, not a conventional corporate landing page.
